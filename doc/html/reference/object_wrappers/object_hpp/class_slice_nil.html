Class slice_nil 

class slice_nil;
static const _ = slice_nil();

A type that can be used to get the effect of leaving out an index in a Python slice expression: 

>>> x[:-1]
>>> x[::-1]

C++ equivalent: 

x.slice(_,-1)
x[slice(_,_,-1)]
